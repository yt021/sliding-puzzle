# پازل کشویی (Next.js + TypeScript)

نسخه‌ی بازنویسی شده با ظاهر رترو، تایمر، صدا و کنترل‌های کیبورد. جدول ۳x۳، ۴x۴ و ۵x۵ قابل انتخاب است؛ کاشی‌ها فقط در مجاورت فضای خالی حرکت می‌کنند و پس از حل شدن پیام «YES! You Win!» نمایش داده می‌شود.

## ویژگی‌ها
- ساخت با Next.js 16 و React 19 (TypeScript) بدون بک‌اند.
- شافل قابل حل با بررسی پاریتی وارونگی؛ خانه‌ی خالی در پایین راست حالت حل شده قرار می‌گیرد.
- کنترل ماوس و کیبورد (کلیک یا Arrow / WASD، اینتر/اسپیس برای حرکت کاشی‌های مجاور).
- تایمر با چشمک ملایم حین بازی، انیمیشن برد و افکت صوتی سه‌مرحله‌ای.
- اندازه‌ی کاشی‌ها نسبت به اندازه‌ی برد برای ۳x۳، ۴x۴ و ۵x۵ مقیاس می‌شود.

## شروع سریع
```bash
npm install
npm run dev
# سپس در مرورگر: http://localhost:3000
```

بیلد و اجرا در حالت پروداکشن:
```bash
npm run build
npm start
```

## روش بازی
1) روی «Start» بزنید تا جدول شافل و تایمر شروع شود.  
2) فقط کاشی‌های کنار خانه‌ی خالی قابل کلیک/فوکوس هستند.  
3) وقتی ترتیب ۱ تا آخر و خانه‌ی خالی در گوشه‌ی پایین راست قرار گرفت، پیام برد نمایش داده می‌شود و تایمر متوقف می‌شود.  
4) برای بازی مجدد روی «Reset» بزنید.

## ساختار پروژه
- `app/page.tsx`: ترکیب صفحه شامل هدر، برد، کنترل‌ها.
- `app/components/*`: اجزای نمایشی (`TimerHeader`، `Board`، `Tile`، `Controls`).
- `app/hooks/usePuzzle.ts`: منطق بازی، تایمر، فلش‌ها، شافل قابل حل.
- `app/hooks/useInterval.ts`: هوک تکرار شونده.
- `app/lib/puzzle.ts`: توابع کمکی پازل (همسایه‌ها، پاریتی وارونگی، شافل قابل حل، حل‌کننده).
- `app/lib/audio.ts`: صداهای حرکت و برد.
- `public/Assets/`: لوگو و فایل‌های صوتی.
- `index.html`, `style.css`, `index.js`: نسخه‌ی استاتیک اولیه (صرفا برای مرجع).

## تست‌ها
اجرای تست‌های کمکی پازل با Vitest:
```bash
npm test
```

## نکات
- حل‌کننده برای ۳x۳ تا ۵x۵ فعال است؛ برای اندازه‌های بزرگ‌تر توصیه نمی‌شود.
- اگر به صدا نیاز ندارید، می‌توانید فایل‌های `Assets/sounds` را حذف یا بی‌صدا کنید.
